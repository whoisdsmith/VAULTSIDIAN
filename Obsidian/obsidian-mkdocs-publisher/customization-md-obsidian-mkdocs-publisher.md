# Customization.md - Obsidian Mkdocs Publisher

---
-   You can prevent the script to share file in specific folder, with editing `folder` list in `exclude.yml`
-   You can prevent the script to **delete** some file with editing `file` list in the same file.
-   You can, also, create some CSS customization with hashtag, with editing `docs/assets/css/custom_attributes.css`. See [Template/customization](https://obsidian-publisher.netlify.app/Template/customization/#custom-attribute "Template/customization") for some example.
