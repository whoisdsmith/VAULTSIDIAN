# How  to install Nano Kontrol Studio 1 in Reaper

1. Install software in this order

- [Korg driver](https://www.korg.com/us/support/download/)
- Attachments [here](https://forums.cockos.com/showthread.php?p=2005577), following the instructions in the third topic.

2. Set NanoKONTROL Studio 1 CTRL - Control Only in Reaper prefs -> MIDI Devices -> MIDI Inputs 
3. Set NanoKONTROL Studio 1 CTRL - Enabled in Reaper prefs -> MIDI Devices -> MIDI Inputs

Check this folder for backup files.
