# chatgpt-md-templates

A companion for [ChatGPT MD](https://github.com/bramses/chatgpt-md).

## Usage

Templates should be stored in the folder specified in ChatGPT MD Settings > Chat Folder Template

To use:

1. Add a title to the file
2. Create a new chat from template with the Command Prompt: Cmd + P > `Create New Chat From Template`
