# Introduction

Support Nature, Science, IEEE, ACS, Web of Science, Springer, Microsoft Academic, Baidu xueshu, scinapse

To support new websites, you have several choices.

-   Leave a comment
-   Create an issue on [GitHub](https://github.com/ChaseChoi/doi2scihub/issues)
-   Add the link to **User matches** manually. For example, Tampermonkey users could do like this: Dashboard > Installed userscripts > DOI to Sci-Hub > Settings > Includes/Excludes > User matches > Add...

# Support user-defined base URL

Click `Settings` button to configure custom base URL

![settings](https://raw.githubusercontent.com/ChaseChoi/doi2scihub/main/Assets/access-settings-page.png)

## Add custom base URL

![set-base-url](https://raw.githubusercontent.com/ChaseChoi/doi2scihub/main/Assets/set-custom-base-URL.png)

1.  Enter custom base URL
2.  Click `Save` button
3.  Click `Close` button to close settings page

## Remove custom base URL

![reset-base-url](https://raw.githubusercontent.com/ChaseChoi/doi2scihub/main/Assets/reset-custom-base-URL.png)

1.  Click `Reset to defaults` button
2.  Click `Save` button
3.  Click `Close` button to close settings page
