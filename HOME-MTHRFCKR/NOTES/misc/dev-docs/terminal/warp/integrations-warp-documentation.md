# Integrations - Warp Documentation

> ## Excerpt
> To attach this configuration to a keyboard shortcut, you must go to the Apple Menu -> Preferences. Then go to Keymap -> External Tools. You will find Open Warp. Right click on it, and select Add Keyboard Shortcut. Type your desired shortcut and voila! You're ready to open Warp with a keyboard shortcut.

---
To attach this configuration to a keyboard shortcut, you must go to the Apple Menu -> `Preferences`. Then go to `Keymap` -> `External Tools`. You will find `Open Warp`. Right click on it, and select `Add Keyboard Shortcut`. Type your desired shortcut and voila! You're ready to open Warp with a keyboard shortcut.
