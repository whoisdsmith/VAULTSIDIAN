# A.I. Command Search - Warp Documentation

> ## Excerpt
> Press CTRL-` to open the AI Command Search panel. Type in the input box what you'd like to do and press ENTER to generate the command. Once the result shows up, press CMD-ENTER to input the generated command directly into Warp's Input Editor or modify the command description and ENTER to re-generate (make another query).

---
Press `` CTRL-` `` to open the AI Command Search panel. Type in the input box what you'd like to do and press `ENTER` to generate the command. Once the result shows up, press `CMD-ENTER` to input the generated command directly into Warp's Input Editor or modify the command description and `ENTER` to re-generate (make another query).
