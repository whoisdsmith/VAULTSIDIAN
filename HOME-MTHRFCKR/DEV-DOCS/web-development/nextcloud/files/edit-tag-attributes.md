# Edit tag attributes

 Usage

 `tag:edit [--name [NAME]] [--access [ACCESS]] [--] [<id>]`

edit tag attributes

 Arguments

 `id`

The ID of the tag that should be deleted

 Is required: no

 Is array: no

 Default: `NULL`

 Options

 `--name`

sets the 'name' parameter

 Accept value: yes

 Is value required: no

 Is multiple: no

 Default: `NULL`

 `--access`

sets the access control level (public, restricted, invisible)

 Accept value: yes

 Is value required: no

 Is multiple: no
