## Module Installers

_Valid for versions 82 through the latest version_

#### Version:

#### [82](https://docs.cpanel.net/whm/software/module-installers/)

___

Last modified: _September 29, 2020_

This WHM feature provides access to four interfaces that you can use to find and install modules for PHP, Ruby, and Perl. You can use the modules that you install, for example, in custom scripts.

-   [PHP PECL](https://docs.cpanel.net/whm/software/php-pecl/) — Install and manage PHP extensions from [PECL](http://pecl.php.net/) (PHP Extension Community Library).
-   [Ruby Gem Installer](https://docs.cpanel.net/whm/software/ruby-gem-installer/) — Install and manage Ruby Gems from [RubyForge](https://rubygems.org/gems/rubyforge).
-   [PHP Extensions and Applications Package](https://docs.cpanel.net/whm/software/php-extensions-and-applications-package/) — Install and manage PHP extensions from [PEAR](http://pear.php.net/) (PHP Extensions and Applications Repository).
-   [Install a Perl Module](https://docs.cpanel.net/whm/software/install-a-perl-module/) — Install and manage Perl modules from [CPAN](http://www.cpan.org/) (The Comprehensive Perl Archive Network).
