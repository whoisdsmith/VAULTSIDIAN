# Expires the users trashbin

 Usage

 `trashbin:expire [<user_id>...]`

Expires the users trashbin

 Arguments

 `user_id`

expires the trashbin of the given user(s), if no user is given the trash for all users will be expired

 Is required: no

 Is array: yes

 Default: `array ()`
