# Create a UX diagram from any app in seconds

cover: https://screenshotflow.com/images/websitescreenshot.png
created: November 23, 2022 6:57 PM (UTC)
description: Screenshot Flow is a mobile app that auto-generates user flow diagrams from screenshots to help communicating an app's design with your team or to create visual instruction guides. The flows can be uploaded effortlessly to Google Drive and the Screenshot Flow website provides a light viewer for quick & easy sharing. The diagrams files are fully compatible with draw.io the online diagramming tool for almost limitless possibilities of further editing.
