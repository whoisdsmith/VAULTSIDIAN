## How to Setup Ssh and Clone Github Repository through Ssh?

1. Create ssh - key  
``` ssh-keygen -t rsa ```
2. copy the generated RSA content to ssh settings in github dashboard.
3. eval `ssh-agent`
4. ssh-add <your-rsa-file-name>
