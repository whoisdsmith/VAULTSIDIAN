**Last updated:** February 17, 2021

You can use **Translate by Zapier** to detect languages or translate words, phrases, and web pages into over 100 languages.

Translate by Zapier can only translate 1,000 characters at a time.

___

## [1\. Add a translation step to your Zap](https://zapier.com/help/create/format/translate-text-in-zaps#add-a-translation-step-to-your-zap)

___

## [2\. Option 1: Set up your "Translate Text" step](https://zapier.com/help/create/format/translate-text-in-zaps#option-1-set-up-your-translate-text-step)

If you need to update the _Target Language_ field dynamically, you can click the **Custom** tab and select a **field** from a previous step. Translate by Zapier uses the [two-letter ISO 639 format](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) for language codes.

If your previous step encodes the language in another format, you can add a [lookup table](https://zapier.com/help/create/format/create-lookup-tables-in-zaps) to match the language to the corresponding ISO 639 code.

___

## [3\. Option 2: Set up your "Detect Language" step](https://zapier.com/help/create/format/translate-text-in-zaps#option-2-set-up-your-detect-language-step)

Once you've set up the Translate step, you can use the result of the step in further [actions](https://zapier.com/help/create/basics/set-up-your-zap-action) in your Zap.
