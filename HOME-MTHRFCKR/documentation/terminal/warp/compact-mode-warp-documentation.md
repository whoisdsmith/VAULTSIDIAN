# Compact Mode - Warp Documentation

> ## Excerpt
> Warp offers the option to enable compact mode, which condenses the spacing between blocks enabling more content to be in view.

---
Warp offers the option to enable compact mode, which condenses the spacing between blocks enabling more content to be in view.

Compact mode can be toggled through the app menu -> Warp -> Compact Mode. Like theme settings, this selection will persist across sessions.
