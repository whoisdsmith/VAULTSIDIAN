Note

```
Requests for permanent registration must be preceded by mailing list review, 
per Section 7.2 of [RFC7595].
    
```

Note

```
Schemes for which the primary reference is an Internet-Draft are generally 
considered to be provisional until the I-D is approved by the IETF for RFC 
publication, even when the specification requests permanent registration.
    
```
